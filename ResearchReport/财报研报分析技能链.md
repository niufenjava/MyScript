# 财报·研报·调研报告 分析技能链

> 技能组合：PDF读取 → 内容提取 → 智能摘要 → 结构化输出 Markdown

---

## 已安装技能总览

| 技能 | 用途 | 安装量 |
|---|---|---|
| `pdf` | PDF 文本/表格提取（内置） | - |
| `docx` | Word 文档读取（内置） | - |
| `pdf-to-markdown` | PDF 转格式化 Markdown | 594 |
| `document-summary` | 智能摘要框架 | 189 |
| `china-stock-analysis` | A股财务分析/估值 | 11.2K |
| `stock-announcements` | A股公告实时查询 | - |

---

## 技能链说明

### Step 1 — 读取文档（按文件类型选）

```bash
# PDF 文件（原生文字）
node ~/.agents/skills/pdf-to-markdown/scripts/convert.cjs \
  /path/to/财报.pdf --mode native --output ./output/

# PDF 文件（扫描件/图片型，需要 tesseract OCR）
node ~/.agents/skills/pdf-to-markdown/scripts/convert.cjs \
  /path/to/研报.pdf --mode ocr --output ./output/

# Word 文件 → 直接用内置 docx skill
```

**或直接丢给我路径**，我调用 `pdf` / `docx` skill 读取。

### Step 2 — 智能摘要分析

使用 `document-summary` skill 的输出格式框架：

```markdown
## Summary
[2-3句话概述：这是什么报告、主要结论]

## 关键发现
- 发现1：[原文引用/数据]
- 发现2：[原文引用/数据]
- 发现3：[原文引用/数据]

## 核心数据
| 指标 | 数值 | 同比/环比 |
|---|---|---|
| 营收 | X亿 | +Y% |
| 净利润 | X亿 | +Y% |
| 毛利率 | X% | +Ypp |

## 风险提示
- 风险1：[依据]
- 风险2：[依据]

## 投资观点（针对研报）
[买入/持有/卖出] 理由：
1. ...
2. ...
```

### Step 3 — A股专项分析（仅限财报）

结合 `china-stock-analysis` skill，传入股票代码，输出：
- 杜邦分析
- 现金流健康度
- 资产负债异常检测
- 行业对比
- 综合估值评分

### Step 4 — 研报监控（可选）

用 `stock-announcements` skill 查询公司最新公告，
判断与研报逻辑是否一致：
- 业绩是否符合预期
- 重大事项是否影响投资逻辑

---

## 使用方式

把文件路径或 URL 发给我，告诉我文档类型：

- **"帮我分析这份研报"** → `pdf-to-markdown` + `document-summary`
- **"帮我看这个财报"** → `pdf` + `china-stock-analysis` + `document-summary`
- **"帮我查这公司的最新公告"** → `stock-announcements`
- **"出一份完整的分析报告"** → 上述技能链联动，输出 Markdown
