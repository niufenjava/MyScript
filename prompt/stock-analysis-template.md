## 📈 维度二：技术面分析

调用 skill: stock-analyzer

### 基础分析

1. 趋势判断：当前趋势（上升/下降/震荡）
2. 关键位置：支撑位、压力位、当前价位置
3. 主要指标：MACD（交叉信号）、RSI（超买超卖）、MA5/MA10/MA20
4. 成交量分析：展示近20个交易日成交量变化以及近期量能异动分析（每一个交易日都要展示倒序，缩量的时候用明显的图标标记下）：日期 涨幅	开盘	最高	最低	收盘	成交量	异动
5. 均线系统：多头/空头排列状态

### ⏱️ 短线操作价位（分析偏好为"短线交易"或"波段操作"时必须输出）

6. **买入区间**：建议买入价格区间 + 理想买点（结合支撑位 + RSI超卖信号）
7. **卖出区间**：建议卖出价格区间 + 理想卖点（结合压力位 + RSI超买信号）
8. **止损位**：严格止损价格（买入价下方 3%~5% 或跌破关键支撑）
9. **止盈位**：分批止盈点位（第一目标位 / 第二目标位）
10. **盈亏比**：止损幅度 vs 止盈幅度，计算最低盈亏比要求（建议 ≥ 2:1）

Skill 描述：

- 综合评分维度：价值 35% / 技术面 25% / 成长性 25% / 财务健康 15%
- 支持 A股（沪深）、美股、港股
- 输出包含可执行的交易策略（长线持有/波段/短线）
- 数据源：Yahoo Finance
- 短线价位基于：支撑压力位 + RSI + MACD 交叉 + Bollinger Bands 综合判断
