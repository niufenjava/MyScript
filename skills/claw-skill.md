# 技能总览

本文档汇总所有可用的 Agent Skills，按功能领域分类，每个技能用极简方式说明。

---

## 一、金融与股票

### stock-announcements
上市公司公告查询工具。支持按股票代码、关键词、时间范围筛选公告，可提取 PDF 附件内容。适用于快速查阅个股公告信息。
如果想查历史公告，这个技能目前做不到——需要改脚本逻辑，比如循环多个日期再合并结果。

### akshare-stock
A股量化数据分析工具，基于 AkShare 数据源。提供实时行情、历史K线、财务数据、板块分析、资金流向、龙虎榜、新股IPO、融资融券等全面数据。适合需要进行量化分析和市场监控的投资者。

### stock-analyzer
股票深度分析工具，生成专业级投资报告。支持 A股/美股/港股，从实时行情、价值投资指标、技术指标、成长性、财务健康等 8 个维度进行综合评级。数据源为 Yahoo Finance，免费使用。

### stock-explorer
Yahoo Finance 数据查询工具。支持实时报价、基本面摘要、ASCII 趋势图、专业技术分析报告、综合报告等功能。无需 API Key，适合快速查询行情和技术分析。

### china-stock-analysis
基于价值投资理论的 A 股分析工具。提供股票筛选、财务深度分析、行业横向对比、估值计算（含安全边际）等模块。面向低频交易的普通投资者，免费使用。

## qing-skills@technical-analysis
安装 qing-skills@technical-analysis + 用你的 parquet 做单股技术分析

## 二、文档与数据处理

### docx
Word 文档处理工具。支持读写编辑转换 Word 文档，可提取文字转 Markdown、创建合同报告、保留修订痕迹、转 PDF/图片。适用于合同协议、文档修改、批量格式处理等场景。

### xlsx
电子表格处理工具。支持创建、编辑、分析电子表格，含公式计算、数据可视化、格式设置等功能。适用于 .xlsx、.xlsm、.csv、.tsv 等格式。

### data-analysis
Excel/CSV 数据分析工具。支持多工作表处理、聚合过滤、数据透视表、SQL 查询，结果可导出为 CSV/JSON/Markdown。适用于上传表格后的结构化数据探索和分析。

### pdf
PDF 操作工具包。支持提取文本和表格、创建新 PDF、合并拆分文档、填写处理表单。适用于批量处理、生成或分析 PDF 文档。

## 三、网络与浏览器自动化

### web-search
实时网络信息搜索工具。通过真实浏览器搜索最新新闻、技术文档、行业动态，支持网页访问和截图。适用于查最新资料、搜错误解决方案、比价调研等场景。

### playwright
浏览器自动化命令行工具。通过终端命令控制浏览器进行点击、填表、截图、数据提取等操作。比 web-search 更底层，适合需要精确控制浏览器行为的自动化任务。

### agent-browser
浏览器自动化 CLI 工具。支持页面导航、表单填写、按钮点击、截图、数据提取、网页测试等功能。触发词包括"打开网站"、"填写表单"、"登录网站"等。

## 四、信息获取

### daily-trending
今日热搜抓取工具。从微博、知乎、百度、36氪、虎嗅、澎湃等平台获取实时热搜，自动过滤娱乐八卦和软广，输出 5 条最有价值的热点。直接说"今日热搜"即可触发。

### weather
天气查询工具。获取当前天气和天气预报信息，无需 API 密钥，免费使用。

## 五、系统与工具

### local-tools
本地系统资源访问工具。支持 macOS 和 Windows 平台的日历访问与管理，适用于直接在设备上管理日程安排。

### 1password
密码管理工具。支持安装和配置 1Password CLI，进行单账户/多账户登录，通过 op 命令读取、注入和运行密钥。

### desktop-control
桌面自动化工具。支持鼠标、键盘和屏幕控制，实现高级桌面自动化操作。

---

## 六、技能管理

### skill-creator
技能开发工具。支持从零创建新技能、更新优化现有技能、运行评估测试、性能基准分析。适用于技能作者进行技能全生命周期管理。

### find-skills
技能发现工具。当用户询问"我如何做 X"、"找一个 X 的技能"时，帮助发现和安装智能体技能。适用于寻找可安装技能的扩展功能场景。

### clawhub
技能包管理工具。通过 ClawHub CLI 从 clawhub.com 搜索、安装、更新和发布技能。支持版本管理和技能文件夹发布。

## 七、AI 与提示工程

### bootstrap ✅
AI 个性配置工具。通过引导对话生成个性化的 SOUL.md 文件，用于创建或更新 AI 伙伴的身份设定。
触发词包括"创建我的 SOUL.md"、"更新我的 AI 个性"等。

### prompt-engineering-expert ✅
提示词工程专家工具。提供提示词优化、自定义指令设计、AI 代理提示词改进等高级功能。适用于提升 AI 交互质量和任务执行效果。

### self-improving-agent ✅
一句话： 把工作中的教训、错误、功能需求记录下来，让 AI 越用越聪明。
注意：让AI帮你记录到心跳里面。

### Self-Improving + Proactive Agent ✅
这是一个自我改进 + 主动出击型 Agent技能，核心目标是让 AI 自动从错误和偏好中学习，知识随时间累积。
这个技能对我（你的 AI 搭档）来说非常有用——它解决的是一个常见痛点：每次新会话 AI 都"失忆"，要重新认识用户。有了这个，偏好和习惯会跨会话累积，真正做到越用越懂你。
错误后自动触发自我修正
