# MyClawScript

这是一个用于 Claw 相关工作的项目仓库。

## 项目用途

1. **编写 Claw 相关的 Python 脚本**
   - 位于 `stock/` 等目录中
   - 用于数据处理、分析等功能

2. **记录 Claw 相关的提示词（Prompt）**
   - 位于 `prompt/` 目录中
   - 包含各类任务的 prompt 模板和 skill 定义

## 目录结构

```
.
├── prompt/         # 提示词模板和 skill 文件
├── stock/          # Python 脚本和相关数据
└── ...
```
